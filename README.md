# App Visual 01

Simple and clean design to a wallet app. 

## Installation

Use the package manager npm to install de dependencies:

```bash
npm install
```

Or the yarn:

```
yarn 
```

For android phones: 

- Start the metro server
```
react-native start
```

- Install the android application
```
react-native run-android
```
